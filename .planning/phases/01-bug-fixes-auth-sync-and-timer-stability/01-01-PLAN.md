          const { data: sessionData } = await supabase.auth.getSession();

| T-01-02 | Information Disclosure | localStorage auth token (sb-lpnjxkzeyiifzzycmftg-auth-token) | accept | Token stored by Supabase with `persistSession: true`. This is standard practice for Supabase auth. Token is a JWT with limited scope. |
